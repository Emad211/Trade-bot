This directory is the durable research memory for the second-generation trading-system project. Earlier model, market, asset, and timeframe choices are not inherited unless this program re-establishes them with evidence.

## Program objective

Build a defensible path from economic hypotheses to a live system without confusing source availability, engineering success, prediction accuracy, historical backtests, hidden risk premia, or public visibility with exploitable edge.

## Governance

- Current metadata is never projected backward.
- Missing provider timestamps remain missing.
- Source failure is not a market null and is never filled by carry-forward or interpolation.
- Public accessibility is not permission for retention, redistribution, combination or derived returns.
- Engineering passes are not empirical passes.
- Paper replication passes are not economic-edge passes.
- Negative results and access blockers remain permanent research evidence.
