# Phase 3G — prospective market/semantic overlap plan

Phase 3G will collect a current, public, two-venue BTC 4h market snapshot and align it
with the verified longitudinal semantic state. It will measure real prospective
overlap and maturity growth without fitting a model or creating a trading decision.

The implementation is tracked in Issue #30 and will be developed on
`agent/phase-3g-prospective-overlap`.
